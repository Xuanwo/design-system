Status / category label. Default `subtle` (tinted) reads quietly inline; `solid` for emphasis. Use `dot` for live status.

```jsx
<Badge color="accent">New</Badge>
<Badge color="success" dot>Stable</Badge>
<Badge color="warning" variant="solid">Beta</Badge>
```

Colors: `neutral` / `accent` / `success` / `warning` / `danger` / `info`. Not clickable — for removable, interactive chips use `Tag`.
