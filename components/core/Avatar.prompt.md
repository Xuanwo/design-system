Author / user avatar. Shows the image, or azure initials if there's no `src`.

```jsx
<Avatar name="Xuanwo" />
<Avatar src="/me.jpg" name="Xuanwo" size="lg" />
<Avatar name="Han Xiao" size="sm" status="online" />
```

Sizes `xs`–`xl` or a raw px number. `shape="rounded"` for app/product contexts, `circle` (default) for people.
